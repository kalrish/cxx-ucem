template<class CodedCharacterSet1 , class CodedCharacterSet2 >
CodedCharacterSet2::nonnegative_integer Unicode_CharacterEncodingModel::translate_code_point ( typename CodedCharacterSet1::nonnegative_integer  i)
noexcept

Translate a code point between two coded character sets.

Attention
Do not confuse translating with transliterating. What this function does is translate a code point from one coded character set to another if both coded character sets map it. The latter is outside the scope of this project, and thus is not handled at all by this library.
Template Parameters
CodedCharacterSet1Input coded character set.
CodedCharacterSet2Output coded character set.
Parameters
[in]iA nonnegative integer mapped by both the input and output coded character sets.
Exceptions
Unicode_CharacterEncodingModel::untranslatableIf the character associated to the given nonnegative integer by the source coded character set is not mapped by the target coded character set.
Returns
The nonnegative integer that the output coded character set maps to the character mapped to the input nonnegative integer by the input coded character set.
