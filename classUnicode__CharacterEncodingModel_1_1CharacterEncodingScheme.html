Detailed Description

template<std::size_t CodeUnitWidth>
class Unicode_CharacterEncodingModel::CharacterEncodingScheme< CodeUnitWidth >

Base class for character encoding schemes.

Character encoding scheme implementations should not derive from this class; instead, they should derive from the class template appropriate to their type.

Template Parameters
CodeUnitWidthThe binary width (number of bits) occupied by each code unit.

template<std::size_t CodeUnitWidth>
using Unicode_CharacterEncodingModel::CharacterEncodingScheme< CodeUnitWidth >::code_unit = typename detail::select_appropriate_code_unit<detail::round_to_byte_multiple(CodeUnitWidth)>::type

Integer type capable of holding any of the values that code units mapped by the character encoding scheme may take.

Note
This type might be capable of holding values not handled by the character encoding scheme. For example, on a system where bytes had 8 bits each, a character encoding scheme that worked with code units 7-bit wide would get a type with at least 8 bits.
