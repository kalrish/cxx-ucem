Detailed Description

template<class Derived, unsigned long long Min, unsigned long long Max>
class Unicode_CharacterEncodingModel::CodedCharacterSet< Derived, Min, Max >

Base class for coded character sets.

Coded character set implementations should derive from this class.

static

Maximum nonnegative integer that the coded character set maps.

Attention
The set of nonnegative integers mapped by a coded character set may not be a contiguous range. Therefore, it would be wrong to check whether a nonnegative integer is mapped by a coded character set by checking whether it is greater than or equal to the minimum value mapped and equal to or less than the maximum value mapped; instead, use maps.
